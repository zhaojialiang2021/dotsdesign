# /spec {name}

面向开发。读取组件文档和令牌表，输出开发的精确参数——间距、颜色、圆角、动效曲线、触觉波形。不需要开发在 Figma 里量间距、截图对比色值。

## 前置条件

- 目标组件/页面文档 status 必须为 complete 或 draft

## 执行流程

### 1. 读取文档（强制）

1. `references/design-tokens.md` — 解析所有令牌值
2. `references/design-principles.md` — 理解规则和约束
3. `references/haptics.md` — 触觉波形参数
4. 目标组件/页面文档

### 2. 解析规格

从组件文档的规格表和 Paper 代码段提取所有数值：

#### 尺寸
- 宽/高
- 最小触控目标（必须 >= 44px）

#### 间距
- 内边距（padding），映射到具体 px 值
- 外边距/间距（gap），映射到具体 px 值

#### 颜色
- 每个元素的颜色令牌名 → Light 值 → Dark 值
- CSS 变量写法：`var(--token-name)`

#### 圆角
- 令牌名 → px 值
- CSS 变量写法：`var(--token-name)`

#### 字号
- 令牌名 → 大小 / 字重 / 行高

#### 动效
- 时长、曲线

#### 触觉
- 交互 → 意图名 → 波形文件路径

### 3. 输出参数清单


### 4. 严选确认

对每行输出值问自己：
- 这个值确实来自令牌文档吗？（不是凭记忆）
- 这个令牌确实存在于 design-tokens.md 吗？

任何不确定的值标注「⚠️ 未验证」而非猜测。
