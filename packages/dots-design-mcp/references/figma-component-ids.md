## 使用方式

在 use_figma 中创建实例：

```js
// 创建图标实例
const backIcon = (await figma.getNodeById('1916:29046')).createInstance();
// 创建开关（on 态）
const swOn = (await figma.getNodeById('2260:60245')).createInstance();
// 创建 logo
const logo = (await figma.getNodeById('2178:60468')).createInstance();
```

实例创建后仍可调整位置（x/y）、缩放（rescale）和覆盖属性（variantProperties）。
