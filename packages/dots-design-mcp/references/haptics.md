# Dots 触觉反馈令牌

> 触觉反馈是设计系统的一等公民，与颜色、间距同等重要。
> 每个可交互元素必须明确其触觉意图，不能留空。

---

## 意图分级

5 级触觉意图，覆盖所有交互场景。不新增意图。

| 意图 | 体感描述 | 典型场景 |
|------|---------|---------|
| tick | 极轻一点，几乎只是提示 | Tab 切换、选择器滚动、到达边界 |
| light | 轻而清脆的一下 | 按钮点击、收藏/取消、开关切换 |
| medium | 明确的一击，有重量感 | 长按弹出菜单、拖拽放置完成 |
| warning | 两下短促连击，带警示感 | 删除确认、不可操作提示、数据异常 |
| success | 三下渐弱，完成感 | 保存完成、提交成功、操作确认 |

---

## 波形参数

### tick

| 参数 | 值 |
|------|------|
| 强度 | 0.3 |
| 锐度 | 0.4 |
| 持续 | 10ms |

### light

| 参数 | 值 |
|------|------|
| 强度 | 0.5 |
| 锐度 | 0.6 |
| 持续 | 15ms |

### medium

| 参数 | 值 |
|------|------|
| 强度 | 0.7 |
| 锐度 | 0.7 |
| 持续 | 20ms |

### warning（双脉冲）

| 参数 | 第一击 | 静默 | 第二击 |
|------|--------|------|--------|
| 强度 | 0.8 | — | 0.65 |
| 锐度 | 0.8 | — | 0.75 |
| 持续 | 18ms | 62ms | 20ms |

### success（三脉冲渐弱）

| 参数 | 第一击 | 静默 | 第二击 | 静默 | 第三击 |
|------|--------|------|--------|------|--------|
| 强度 | 0.7 | — | 0.5 | — | 0.3 |
| 锐度 | 0.7 | — | 0.6 | — | 0.5 |
| 持续 | 18ms | 40ms | 16ms | 40ms | 14ms |

## 平台导出

每种意图导出为三种格式，参数一一对应，仅单位不同。

| 平台 | 格式 | 路径规则 |
|------|------|---------|
| iOS | `.ahap` (Apple Haptic Audio Pattern) | `haptics/ios/{intent}.ahap` |
| Android | `.he` (RichTap) | `haptics/android/{intent}.he` |
| 鸿蒙 | 振动 JSON | `haptics/harmony/{intent}.json` |

构建脚本自动从本文档提取参数并生成以上三种格式。

## 组件引用方式

组件文档中引用触觉意图：

```
触觉反馈：按钮点击触发 light 意图
```

开发查令牌文档拿到对应平台的波形文件路径和参数，直接接入。

## 防护规则

| 规则 | 说明 |
|------|------|
| 200ms 去重 | 同一手势 200ms 内不重复触发，防止连续振动 |
| 后台禁振 | App 进入后台时禁用所有触觉反馈 |
| 设备检测 | 检测马达类型，无振动马达的设备走视觉反馈降级 |
| 降级策略 | tick/light → 无反馈；medium → 系统默认轻振；warning → 系统通知振动；success → 系统成功振动 |

## 各组件触觉映射

| 组件 | 交互 | 意图 |
|------|------|------|
| Button | 点击 | light |
| TabBar | 切换 Tab | tick |
| Switch | 切换 | light |
| Card | 长按 | medium |
| TypeTag | 点击筛选 | tick |
| 删除操作 | 确认 | warning |
| 保存/提交 | 完成 | success |
| Picker | 滚动 | tick |
| Modal | 弹出 | light |
| PullRefresh | 触发 | medium |
