# Sidebar 侧边栏

> status: draft

从首页左侧滑出的设置面板。

---

## 页面结构

```
侧边栏 (393×852)
├── StatusBar (50px)
├── 内容区 (padding: 72px 16px, 距左 48px)
│   ├── 用户卡片 (`Bg 3`, radius.bubble 22px)
│   │   ├── 头像 (42×42px, 圆形)
│   │   ├── 昵称 (18px Semibold, rgba(0,0,0,0.8))
│   │   └── ID (12px, rgba(0,0,0,0.45))
│   ├── 设置组
│   │   ├── 组标题 "设置" (14px, rgba(0,0,0,0.45))
│   │   └── `Bg 3` 卡片容器 (radius.bubble 22px)
│   │       ├── 通知 (含 Switch 开关)
│   │       ├── 应用语言 → "中文" (含右箭头)
│   │       └── 深色模式 → "跟随系统" (含右箭头)
│   ├── 关于组
│   │   ├── 组标题 "关于" (14px, rgba(0,0,0,0.45))
│   │   └── `Bg 3` 卡片容器 (radius.bubble 22px)
│   │       ├── 帮助中心
│   │       ├── 使用条款
│   │       └── 隐私政策
│   └── 底部版本号 (flex-end, "Mindbank v1.0")
├── 主页面（半遮罩，左侧可见）
└── 遮罩层 (黑色 50% 透明)
```

---

## 视觉规格

- 侧边栏背景：#FCFCFC
- 主页面右侧露出约 48px，被 50% 黑色遮罩覆盖
- 列表项高度 48px
- 列表项文字：Body-Primary (16px/400)，黑色
- 右侧辅助文字：Body-Primary，rgba(0,0,0,0.45)
- 分割线：0.5px Line-NonOpaque
- Switch 开关：24×40px，开启态 #6EC971
- 图标统一 24px，左侧 16px padding

## 交互

- 点击遮罩关闭侧边栏
- 左滑关闭
- Switch 切换：light 触觉
