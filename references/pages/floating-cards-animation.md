# 景深浮动卡片动画

## 需求背景

这个 demo 基于用户提供的录屏缩略帧制作，用于快速验证一类营销页常见的抽象产品动效：软件界面卡片、文档卡片和应用图标在浅色景深背景上缓慢漂浮。

原始 `.mov` 文件实际时长为 2.365 秒，画面里出现的 `0:01 / 0:58` 是录屏中被录进去的播放器控件，不代表文件自身时长。当前 demo 按这段 2.365 秒关键帧还原中间大卡片的焦点切换，不复刻播放器控件。

## 视觉结构

1. 背景：浅灰背景叠加低对比网格，整体带景深虚化。
2. 中景：中心软件面板，白色半透明卡片，内部用工具栏、预览块和表格线模拟产品界面。
3. 前景：横向绿色表格卡，初始处在视觉焦点，随后退到右上远景并虚化。
4. 浮层图标：文档、表格、云服务、消息四个应用入口，使用不同大小、位置和漂浮节奏。
5. 辅助光雾：左右两侧使用低对比色雾制造空间感。

## 动效规则

- 舞台整体入场：透明度从 0 到 1，轻微上移归位。
- 动画主循环为 2.36 秒，匹配当前录屏文件长度。
- 初始态：绿色横向表格卡位于前景焦点，主白色软件面板在后方，透明度较低且有轻微 blur。
- 中段：绿色表格卡向右上退场并逐步虚化；主白色软件面板从后方推到中间，scale 放大，blur 下降。
- 末态：主白色软件面板成为当前焦点；右上文档卡和远端面板保持更低透明度、更强 blur。
- 应用图标跟随景深层移动，不做独立弹跳；位置变化服务于卡片焦点切换。
