# Home 首页

> status: draft

首页是 Dots 的核心页面，采用聊天式 UI，用户与 AI 在同一流中交互。

---

## 页面结构

```
首页 (435×滚动)
├── StatusBar (50px，系统组件)
├── Head (48px)
│   ├── Logo (左侧，104px 宽)
│   └── Star 图标 (右侧，20px)
├── Message 区域 (flex-1，可滚动)
│   ├── 日期标记
│   ├── 消息组（MessageBubble × N）
│   ├── AI 回应区
│   └── ...
└── InputBar (底部固定)
    ├── 附件按钮 (46×46px)
    ├── 输入框 (圆角 100px，含 placeholder + 语音按钮 + 发送按钮)
    └── 附件面板（展开态：摄像头/照片/附件等）
```

---

## Head 导航栏

- 高度 48px，白色背景
- 左侧：Dots 彩色 Logo（104px 宽）
- 右侧：收藏图标（star，20px）
- 无返回按钮，首页是根页面
- 底部无分割线，与内容区无缝衔接

---

## Message 区域

- 页面背景 `Bg`
- 左右边距 16px (Space-4)，消息区域内 padding 16px
- 消息流从上到下按时间排列
- 用户消息靠右对齐（padding-left: 48px）
- AI 回应靠左对齐，全宽
- 同一天的消息归为一组，组间显示日期标记

### 消息流示例

```
[日期标记: 2025/06/11]
[User] 文字消息
[User] 语音消息 + 语音消息（无文字）
[User] 链接卡片
[User] 单图 / 四图
[User] 引用消息
[日期标记: 2025/06/11]
[AI]  带品牌色背景的回应
      引用来源行
[日期标记: 2025/06/11]
[AI]  结构化内容（开场 + 引用块 + 总结标题 + 列表项）
```

---

## InputBar 输入栏

### 默认态

- 白色背景，底部固定
- 底部有渐变遮罩（从透明到白色，用于覆盖内容区底部）
- 渐变高度约 80px

### 布局

```
[附件按钮 46px] [输入框 flex-1] 
```

### 附件按钮

- 46×46px，圆形（Radius-Full），BG-3 背景
- 图标：picture，24px，Label-Secondary

### 输入框

- 高度 48px，Radius-Full (100px)
- 背景 BG-3
- 左侧：placeholder "现在的想法"，Body-Primary，Label-Tertiary
- 右侧：语音图标 (mic, 24px) + 发送按钮 (32px，黑色圆形)

### 附件面板（展开态）

- 从输入框下方弹出
- 横向排列：摄像头 / 照片 / 附件
- 每项：图标 38×38px 圆形容器 + 文字标签
- 图标容器 BG-3 背景，图标 20px

## 滚动行为

- Message 区域可垂直滚动
- 滚动时 Head 保持固定
- InputBar 始终固定在底部
- 新消息到达时自动滚到底部
