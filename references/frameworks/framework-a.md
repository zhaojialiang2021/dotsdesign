# Framework-A: 聊天流

> status: draft

## 描述

Dots 核心交互框架。用户输入 + AI 回应的聊天式对话流，覆盖所有"问 AI 拿答案"的场景。

## 结构

```
┌──────────────────────┐
│      NavBar          │ ← 页面标题 + 返回/更多
├──────────────────────┤
│                      │
│   消息列表（滚动）     │
│                      │
│   ┌──────────────┐   │
│   │ 用户消息气泡   │   │ ← BG-0, 右对齐
│   └──────────────┘   │
│                      │
│   ┌──────────────┐   │
│   │ AI 回应气泡    │   │ ← Brand-Blue-Light 背景 + Brand-Blue-Border 边框, 左对齐
│   │ 由 AI 生成    │   │
│   └──────────────┘   │
│                      │
├──────────────────────┤
│     输入框 + 发送     │ ← BG-0, 底部固定
└──────────────────────┘
```

## 关键规则

- AI 回应气泡：Brand-Blue-Light 背景 + Brand-Blue-Border 边框，标注"由 AI 生成"
- 用户气泡：BG-0 背景 + Line-NonOpaque 边框
- 消息列表自动滚动到最新
- AI 回应 loading 态：骨架屏 + 文字提示"正在思考…"
- AI 回应失败态：错误提示 + 重试按钮

## 多宽度适配

| 断点 | 变化 | 说明 |
|------|------|------|
| Mobile (390px) | 无变化 | 标准体验 |
| Tablet (768px) | 消息最大宽度 540px，居中 | 侧边栏同时可见 |
| Desktop (1440px) | 消息最大宽度 680px，居中 | 左侧导航栏固定 |
